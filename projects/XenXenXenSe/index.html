XenXenXenSe

A Modern RESTful API implementation of XenAPI for XCP-ng and Citrix Hypervisor速.

 

Main problems we face during using XCP-ng in production is hard-to-implement feature of XMLRPC based XenAPI. Xen Orchestra does provide JSON-RPC based solution, but it is not intuitive enough as competitor like VMWare is.

XenXenXenSe is a software solution that solves those issue by providing stable HTTP based API Interface between XenAPI and developing application
